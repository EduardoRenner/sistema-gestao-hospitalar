# Sistema de Gestão Hospitalar

## Descrição
Projeto desenvolvido em Java com integração a banco de dados SQLite, implementando um CRUD completo de pacientes.

## Funcionalidades
- Cadastro de pacientes
- Listagem de pacientes
- Atualização de dados
- Remoção de pacientes

## Tecnologias
- Java
- JDBC
- SQLite

## Estrutura
- DAO (Data Access Object)
- Service
- Model

## Como executar
1. Clone o repositório
2. Abra na IDE
3. Execute o projeto
